# Development Conference 2026

A responsive landing page for a fictional developer conference, built with pure HTML and CSS.

## 🔗 Links

- **GitHub Repository:** https://github.com/mijbahur/b14-assignment-01-PH
- **Live Website:** https://mijbahur.github.io/b14-assignment-01-PH/

## 📄 About

This project is a single-page website for "Development Conference 2026," a fictional 3-day event for engineers, founders, and builders. It includes:

- **Navbar** – Fixed navigation with smooth-scroll anchor links to each section
- **Hero Section** – Full-width banner with headline, description, and call-to-action
- **Speakers** – Grid of featured speaker cards with photo, category tag, and title
- **Pricing** – Three-tier pricing cards (Standard, Pro, Team) with a highlighted featured plan
- **Venue** – Location details with image and directions link
- **FAQ** – Expandable question/answer list using native HTML `<details>` elements
- **Footer** – Social links, copyright, and legal links

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid for responsive scaling)
- [Font Awesome](https://fontawesome.com/) for social icons
- [Google Fonts](https://fonts.google.com/) (Inter)

## 📌 Notes

This is a practice/coursework project built to strengthen CSS layout, responsiveness, and interactivity skills without JavaScript.
